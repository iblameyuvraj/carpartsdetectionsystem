* Tailwind CSS
* Shadcn UI
* Magic UI
* Aceternity UI
* Framer Motion

## 🛠️ Installation
To run Orbit locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/iblameyuvraj/carpartsdetectionsystem
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Run the development server:
    ```bash
    npm run dev
    ```
